vimコマンド
移動
h j k l
gg : 先頭へ
G  : 最終行へ
5G : 5行目へ
w  : 単語の先頭へ
e  : 単語の最後へ
b  : 前の単語へ
%  : 対になる括弧へ
cntl + e | y / cntl + u | d : スクロール

編集
i  : その場で挿入モード
I  : 行の最初で挿入モード
a  : 一文字後ろで挿入モード
A  : 行の最後で挿入モード
o  : 下に改行して挿入モード 
O  : 上に行を入れて挿入モード

cw : 単語を消して挿入
c$ : カーソル以降を消して挿入
c% : 括弧内を消して挿入

p  : ペースト
P  : 前にペースト
yy : コピー
dd : カット
u  : アンドゥ
cntl + r : リドゥ

r  : 一文字置換
R  : 複数文字置換

ビジュアル
=  : インデント揃え

全般
:tabnew : 新規タブの作成
:tabe   : ファイルを開く
gt      : タブ移動
:tabclose : タブを閉じる
